# Job Application Portal (HTML Project)

# Project Description
        This project is multi-page Job Application Portal built using pure HTML.
        It allows users to explore job opportnities, create an account, log in, and apply for jobs through a structured application form.

# Features
    ## Homepage with:
        > Company Introduction
        > List of available job positions

    ## Job Application Form:
        > Personal Information(Radio buton)
        > Education Details(Dropdown)
        > Skills Selection(Checkboxes)
        > Work Experience
        > Resume Upload
        > Additional Information (LinkedIn, Github, Portfolio,etc)
        > Position Selection(Dropdown)

    ## Authentication Pages:
        > Sign Up Page
        > Login Page

    ## Multipage navigation using HTML links
    ## Clean and structured semantic HTML
