    # Lavender's Kitchen Website 🍽️

    ## Overview
        Lavender's Kitchen is a simple, multi-page restaurant website built using HTML.  
        It demonstrates a professional website structure, navigation between pages, and basic forms for user interaction.

        This project is perfect for beginners to practice HTML, website layout, and content organization.

    ## Pages Included

        | Page | Description |
        | (index.html) | Home page with welcome message and "Why Choose Us" section |
        | (menu.html) | Displays menu items: Appetizers, Main Courses, Desserts |
        | (about.html) | About Us page with story, values, and team information |
        | (gallery.html) | Photo gallery of dishes and restaurant ambiance |
        | (contact.html) | Contact information and reservation form |
        | (signup.html) | Sign Up page for users to create an account |
        | (login.html) | Login page for returning users |

    ## Features
        - Fully structured header and footer
        - Navigation bar linking all pages
        - Reservation form on Contact page
        - Sign Up and Login forms
        - Image gallery with Unsplash image credits
        - Section comments for easy code readability
        - Responsive design (basic mobile-friendly layout via viewport meta tag)
